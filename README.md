# law-office-management-uml-design


This repository contains a UML and software design project for a Law Office Management System.

The project includes functional and non-functional requirements, behavioral UML diagrams, structural UML diagrams, CRC cards, and CRUD analysis. It demonstrates experience with requirements analysis, object-oriented design, software modeling, and software engineering documentation.

# All diagrams are available in the file "Law_Office_Management_UML_Design_Project.pdf" and only the "Use-Case Description" and "CRC cards" are placed in separate files with the same names.
